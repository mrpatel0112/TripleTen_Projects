# TripleTen_Projects
Projects I worked on while attending the Bussiness Intelligence Analytics TripleTen Bootcamp.


| Project number | Title | Description |
| :-----------: | ----------- |----------- |
| 1 |Manhattan Airbnb Market Analysis | Analyzed Airbnb data using spreadsheets to identify high-demand neighborhoods, popular property sizes, and revenue potential for investment decisions.|
| 2 | E-Commerce Funnel & Cohort Analysis | Analyzed raw user event logs to build conversion funnels and cohort-based retention metrics, helping evaluate customer behavior and purchase retention.|
| 3 |Saving SuperStore – Data Visualization (Tableau) | Analyzed sales, profit, advertising impact, and return rates using Tableau visualizations to identify profit drivers, loss areas, and actionable business decisions.|
| 4 |Superstore Returns Analysis – Storytelling with Data (Tableau) | Analyzed order and return data to identify key drivers of high return rates and built an interactive dashboard to help leadership reduce returned orders.|
| 5 |Shopify App Analysis – Power BI | Analyzed Shopify App Store data to identify key factors driving app success using interactive Power BI dashboards, DAX calculations, and review insights.|
